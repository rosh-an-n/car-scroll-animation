# Car Scroll Animation

a scroll based hero section animation built with next.js and gsap. the car moves across the road as you scroll and reveals the text behind it. stat boxes fade in at different scroll points.

## how to run

```bash
npm install
npm run dev
```

then open http://localhost:3000 in your browser and scroll down.

## built with

- next.js
- gsap (scrolltrigger)
- css
